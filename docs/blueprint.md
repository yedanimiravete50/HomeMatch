# **App Name**: HomeMatch Hub

## Core Features:

- User Authentication and Verification: Secure user authentication with email/social login, and document verification for enhanced trust.
- Property Listings with Interactive Map: Display property listings with detailed filters (city, price, amenities) on an interactive map powered by Mapbox.
- Roommate Matching Algorithm: AI-powered tool to match roommates based on compatibility factors like lifestyle, habits, and preferences.
- Bilateral Review System: Enable students and landlords to review each other after a confirmed stay, ensuring accountability and transparency.
- Messaging and Chat: Real-time chat functionality for direct communication between students and landlords.
- Experiences and Events Feed: Create and join local events (dinners, city tours) to foster community among international students.
- Multilingual Support: Switch between Spanish and English for an inclusive user experience.

## Style Guidelines:

- Primary color: Soft peach (#FFCBA4) to evoke a sense of warmth, youthfulness, and approachability.
- Background color: Light beige (#F5F5DC) for a clean, neutral backdrop.
- Accent color: Pale apricot (#FFE5B4) to highlight key CTAs and interactive elements.
- Body and headline font: 'Inter', a sans-serif for its modern, clean readability and versatility.
- Use Lucide icons for a consistent, clear visual language throughout the platform.
- Employ a 12-column grid for flexible, responsive design with generous whitespace and rounded cards.
- Subtle, smooth transitions and loading animations to enhance the user experience without being distracting.
